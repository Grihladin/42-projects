# 42 Project Badges

This directory contains badge and cover images for projects in the 42
curriculum.

## Features

- Mandatory and bonus project badges
- Numbered C++ module badges
- Cover images for selected projects

## How to use

Browse the `badges/` or `covers/` directory and reference an image from your
README:

```markdown
![Project badge](./badges/libfte.png)
```
