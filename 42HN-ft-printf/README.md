# ft_printf

ft_printf is a C implementation of formatted output inspired by the standard
`printf` function.

## Features

- Supports `%c`, `%s`, `%p`, `%d`, `%i`, `%u`, `%x`, `%X`, and `%%`
- Returns the number of printed characters
- Builds as a reusable static library

## How to run

```bash
cd 42HN-ft-printf
make
```

The generated library is `ft_printf.a`.
