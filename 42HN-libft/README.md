# Libft

Libft is a C library that recreates common standard-library functions and adds
utilities for strings, memory, output, conversion, and linked lists.

## Features

- Character, string, and memory helpers
- Numeric conversion and file-descriptor output
- Linked-list functions in the bonus target

## How to run

```bash
cd 42HN-libft
make
make bonus
```

The generated library is `libft.a`.
