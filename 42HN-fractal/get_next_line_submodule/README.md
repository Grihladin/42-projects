# get_next_line

This is the internal get_next_line copy used by fract-ol.

## Features

- Reads one line at a time from a file descriptor
- Preserves unread data between calls

## How to run

Compile `src/get_next_line.c` and `src/get_next_line_utils.c` with the parent
project.
